To install packages run:
```bash
yarn
```
or
```bash
npm install
```

To start app in dev mode run:
```bash
npm start
```

To build app in dev or production run:
```bash
npm run build:dev
```
or
```bash
npm run build:prod
```
To run builded code run:
```bash
npm run server:dev
```
or
```bash
npm run server:prod
```
